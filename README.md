A Hacker News clone where users can post articles.

## Technologies

- **Rails:** 7.1.3.4
- **Ruby:** 3.0.0

### Installation

1. **Clone the repository:**

    ```sh
    git git@github.com:paulmckissock/Rails-Tutorial.git
    cd Rails-Tutorial
    ```

2. **Install the required gems:**

    ```sh
    bundle install
    ```

3. **Set up the database:**

    ```sh
    rails db:setup
    ```

4. **Start the Rails server:**

    ```sh
    rails server
    ```
